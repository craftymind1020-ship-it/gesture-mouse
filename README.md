# 🖐️ Gesture Controlled Mouse (AI Virtual Mouse)

Control your computer using hand gestures through your webcam.
This project uses computer vision and hand tracking to replace traditional mouse input with intuitive gestures.

---

## 🚀 Features

* 🖱️ Smooth cursor movement using palm tracking
* 👌 Left click using pinch gesture
* ✊ Drag and drop by holding pinch
* 🤏 Right click using thumb + pinky gesture
* 👉 Swipe gestures for:

  * Copy (swipe right)
  * Paste (swipe left)
* 🎯 Gesture filtering for better accuracy
* ⚡ Real-time performance

---

## 🧠 Technologies Used

* Python
* OpenCV (Computer Vision)
* MediaPipe (Hand Tracking)
* PyAutoGUI (Mouse Control)
* NumPy

---

## 🕹️ Controls

| Action      | Gesture                     |
| ----------- | --------------------------- |
| Move Cursor | Move your palm              |
| Left Click  | Quick pinch (thumb + index) |
| Drag        | Hold pinch                  |
| Right Click | Thumb + pinky pinch         |
| Copy        | Swipe right (2 fingers)     |
| Paste       | Swipe left (2 fingers)      |

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/craftymind1020-ship-it/gesture-mouse.git
cd gesture-mouse
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the program:

```bash
python main.py
```
